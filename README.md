# Sentiment Analysis Dashboard

## Overview

This project is a machine learning based Sentiment Analysis Dashboard that analyzes text and predicts whether the sentiment is **Positive, Negative, or Neutral**.
The system uses a trained machine learning model and presents the prediction results through a simple dashboard interface.

The goal of this project is to demonstrate how machine learning models can be integrated into a user-facing application.

---

## Features

* Text sentiment prediction
* Interactive dashboard interface
* Machine learning model integration
* Easy-to-run local setup

## Installation

Clone the repository:

```
git clone https://github.com/SHHIVDEEP310/Sentiment-Analysis-dashboard.git
```

Move into the project folder:

```
cd Sentiment-Analysis-dashboard
```

Install required dependencies:

```
pip install -r requirements.txt
```

---

## Running the Application

Start the application by running:

```
python app.py
```

After running the command, open your browser and go to:

```
http://localhost:5000
```

You will see the dashboard where you can input text and view the predicted sentiment.

---

## Example Usage

Input text such as:

```
I really love this product.
```

Output:

```
Sentiment: Positive
```
